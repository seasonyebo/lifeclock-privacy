Privacy Policy

Effective Date: April 21, 2026

1. Information We Collect

This app collects the following information for lifespan prediction analysis:

  • Name (Korean, Chinese characters — optional)
  • Date and time of birth (optional)
  • Gender
  • Physical information: body type, waist size, height, weight (optional)
  • Health information: smoking, drinking, exercise, sleep, stress, diet, medical history
  • Lifestyle information: residence area, occupation, marital status, children
  • Family information: parents' survival status and age, hereditary medical history
  • Blood pressure, blood sugar (optional — for Premium analysis)

2. Purpose of Collection and Use

Collected information is used solely for providing lifespan prediction services:

  • Epidemiology-based life expectancy calculation
  • Saju and name analysis interpretation (Premium)
  • Monte Carlo simulation (Premium)
  • Check-in health trend analysis

3. Storage and Processing

All entered data is stored and processed only on the user's device.

  • Data is not transmitted to external servers.
  • AsyncStorage (on-device local storage) is used.
  • All data is automatically deleted when the app is uninstalled.
  • No cloud backup or synchronization features are provided.

4. Third-Party Services

This app does not provide users' personal information to third parties by default.

However, when using the Premium AI Insight feature, upon explicit user request, analysis results are sent to Anthropic Claude API to generate explanations.

  • Data transmitted: Anonymized analysis result values only (no personally identifiable information)
  • Transmission: HTTPS-encrypted communication
  • Third-party policy: Anthropic Privacy Policy
  • Usage limits: Self analysis 10 times/month, family 3 times/month, friend 2 times/month

5. Data Retention and Deletion

  • Data is automatically deleted immediately upon app uninstallation.
  • When data is reset within the app (re-entry), existing data is replaced with new data.
  • No separate retention period applies, and as data is not stored on servers, remote deletion does not apply.

6. User Rights

  • Users can delete all data at any time by uninstalling the app.
  • Users can replace existing analysis results with new data through the re-entry feature.

7. Children's Privacy

This app restricts use by children under the age of 14 and does not intentionally collect personal information from children under 14. The age rating is set to 9+.

8. Privacy Officer

  • Company: Studio JS
  • Email: studiojeongsun@gmail.com

9. Changes to This Privacy Policy

If this policy is changed, we will notify you in advance through in-app announcements or this page.

プライバシーポリシー

施行日: 2026年4月21日

1. 収集する個人情報

本アプリは寿命予測分析のため、以下の情報を収集します。

  • 氏名(ハングル・漢字 — 任意)
  • 生年月日、出生時刻(任意)
  • 性別
  • 身体情報: 体型、ウエストサイズ、身長、体重(任意)
  • 健康情報: 喫煙、飲酒、運動、睡眠、ストレス、食習慣、病歴
  • 生活情報: 居住地域、職業、婚姻状況、子供の有無
  • 家族情報: 両親の生存状況・年齢、家族の遺伝病歴
  • 血圧、血糖値(任意 — プレミアム精密分析用)

2. 個人情報の収集および利用目的

収集された情報は寿命予測分析サービスの提供のみに使用されます。

  • 疫学に基づく平均寿命計算
  • 四柱推命・姓名学解釈分析(プレミアム)
  • モンテカルロシミュレーション(プレミアム)
  • チェックイン健康傾向分析

3. 個人情報の保存および処理方法

入力されたすべてのデータはユーザーのデバイス内でのみ保存・処理されます。

  • 外部サーバーへの送信は行いません。
  • AsyncStorage(デバイス内ローカルストレージ)を使用します。
  • アプリ削除時にすべてのデータが自動的に削除されます。
  • クラウドバックアップや同期機能は提供しません。

4. 第三者サービスの利用

本アプリはユーザーの個人情報を原則として第三者に提供しません。

ただし、プレミアムAI解説機能の利用時、ユーザーが明示的に要請した場合に限り、分析結果がAnthropic Claude APIに送信され、解説が生成されます。

  • 送信データ: 匿名化された分析結果の数値のみ(個人識別情報を含まない)
  • 送信処理: HTTPS暗号化通信
  • 第三者ポリシー: Anthropic Privacy Policy
  • 使用制限: 本人分析 月10回、家族 月3回、友人 月2回

5. 個人情報の保有および破棄

  • データはユーザーがアプリを削除すると同時に自動的に破棄されます。
  • アプリ内でデータ初期化(再入力)時、既存データは新規データに置き換えられます。
  • 別途の保有期間はなく、サーバーに保存されないため、遠隔破棄の対象外です。

6. 利用者の権利

  • ユーザーはいつでもアプリを削除することによりすべてのデータを破棄できます。
  • 再入力機能を通じて、既存の分析結果を新しいデータに置き換えることができます。

7. 児童の個人情報保護

本アプリは満14歳未満の利用を制限しており、満14歳未満の児童の個人情報を意図的に収集しません。年齢制限は9+に設定されています。
